# Pre-completion checklist

Before considering the job done, confirm:

- [ ] I ran the acceptance test command and saw the failure first-hand.
- [ ] I changed implementation code, not the test, to fix the cause.
- [ ] No assertions were removed, skipped, or weakened.
- [ ] The full test suite now passes (exit code 0).
- [ ] The change is minimal and scoped to the failure.
